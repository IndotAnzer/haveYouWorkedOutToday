# 健身训练记录应用 - Have You Worked Out Today?

## 项目简介

这是一个功能完整的健身训练记录应用，帮助用户记录每次训练的详细信息，包括训练动作、组数、重量和次数等数据。用户可以发布训练动态，查看其他用户的训练记录，进行点赞和评论互动。

## 技术栈

### 前端
- Vue 3
- Vue Router
- Axios
- Vite

### 后端
- Go 1.20+
- Gin 框架
- GORM ORM 库
- JWT 认证
- MySQL 数据库
- Redis 缓存

## 核心功能

### 1. 用户认证系统
- 注册和登录功能
- JWT 令牌认证
- 权限控制中间件

### 2. 训练记录管理
- 创建训练记录（包含标题、训练心得）
- 添加训练动作（名称、备注）
- 记录动作组数（重量、次数、备注）
- 查看个人训练历史

### 3. 社交互动
- 查看其他用户的训练动态
- 对训练记录进行点赞
- 评论和回复功能

### 4. 数据可视化（规划中）
- 训练数据统计（训练频率、总训练时长、累计重量等）
- 进度图表（力量增长趋势）
- 训练热图（类似GitHub贡献图，显示训练活跃度）

### 5. 个人目标管理（规划中）
- 设定训练目标（增肌、减脂、提高耐力等）
- 目标追踪和提醒
- 成就系统（完成特定训练任务获得徽章）

### 6. 社区互动功能（规划中）
- 关注系统（关注其他用户，查看他们的训练动态）
- 标签系统（为训练添加标签，如#胸肌训练 #有氧训练等）
- 热门训练分享（展示平台上最受欢迎的训练计划）

### 7. 训练计划管理（规划中）
- 预设训练计划（提供常见训练计划模板）
- 自定义训练计划（创建和保存个人训练计划）
- 计划执行提醒

### 8. 智能功能（规划中）
- 训练建议（基于历史数据提供个性化建议）
- 语音记录（支持语音输入训练内容）
- AI训练助手（提供动作指导和纠正建议）

## 安装和运行

### 后端
1. 进入后端目录：`cd backend`
2. 安装依赖：`go mod tidy`
3. 配置数据库连接（修改 `config/config.yml`）
4. 运行应用：`go run main.go`

### 前端
1. 进入前端目录：`cd frontend/test`
2. 安装依赖：`npm install`
3. 运行开发服务器：`npm run dev`
4. 构建生产版本：`npm run build`

## API 接口

### 认证相关
- `POST /api/auth/register` - 用户注册
- `POST /api/auth/login` - 用户登录

### 训练记录相关
- `GET /api/articles` - 获取所有训练记录
- `POST /api/articles` - 创建训练记录
- `GET /api/articles/:id` - 获取单个训练记录
- `PUT /api/articles/:id` - 更新训练记录
- `DELETE /api/articles/:id` - 删除训练记录

### 点赞相关
- `POST /api/likes` - 点赞训练记录
- `DELETE /api/likes` - 取消点赞

### 评论相关
- `GET /api/articles/:id/comments` - 获取训练记录的评论
- `POST /api/articles/:id/comments` - 发表评论
- `POST /api/comments/:commentId/replies` - 回复评论
- `DELETE /api/comments/:commentId` - 删除评论
- `DELETE /api/replies/:replyId` - 删除回复
