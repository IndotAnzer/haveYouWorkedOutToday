# 训练量统计页面设计

## 页面概览

本页面用于展示用户在指定时间段内的训练量统计，核心功能是通过图表展示每个动作的训练组数，帮助用户直观了解自己的训练分布情况。

## 功能模块

### 1. 时间范围选择器
- 开始日期选择器
- 结束日期选择器
- 应用按钮，触发数据加载

### 2. 动作组数统计图表
- 柱状图展示每个动作的训练组数
- 颜色渐变效果，增强视觉体验
- 鼠标悬停显示详细数据
- 支持水平滚动，适应动作数量较多的情况

### 3. 数据详情表格
- 展示每个动作的名称、训练组数、总次数
- 支持按训练组数排序
- 响应式设计，在小屏幕上自动调整布局

## 技术实现

### 前端技术栈
- **Vue 3**：响应式框架
- **ECharts**：数据可视化库
- **Axios**：API请求
- **SCSS**：样式管理

```vue
<script setup>
import { ref, computed, onMounted } from 'vue';
import { use } from 'echarts/core';
import { CanvasRenderer } from 'echarts/renderers';
import { BarChart } from 'echarts/charts';
import { TitleComponent, TooltipComponent, GridComponent, DataZoomComponent } from 'echarts/components';
import VChart from 'vue-echarts';
import axios from 'axios';

// 注册ECharts组件
use([
  CanvasRenderer,
  BarChart,
  TitleComponent,
  TooltipComponent,
  GridComponent,
  DataZoomComponent
]);

// 响应式数据
const startDate = ref(new Date(Date.now() - 30 * 24 * 60 * 60 * 1000).toISOString().split('T')[0]);
const endDate = ref(new Date().toISOString().split('T')[0]);
const actionStats = ref([]);
const loading = ref(false);
const error = ref('');
const sortField = ref('groups');
const sortOrder = ref('desc');

// 计算属性：排序后的数据
const sortedActionStats = computed(() => {
  return [...actionStats.value].sort((a, b) => {
    if (sortOrder.value === 'asc') {
      return a[sortField.value] - b[sortField.value];
    } else {
      return b[sortField.value] - a[sortField.value];
    }
  });
});

// 计算属性：图表配置
const chartOption = computed(() => {
  const actionNames = actionStats.value.map(item => item.action_name);
  const groupCounts = actionStats.value.map(item => item.groups);
  
  return {
    tooltip: {
      trigger: 'axis',
      formatter: '{b}: {c}组'
    },
    grid: {
      left: '3%',
      right: '4%',
      bottom: '3%',
      containLabel: true
    },
    xAxis: {
      type: 'category',
      data: actionNames,
      axisLabel: {
        interval: 0,
        rotate: 45
      }
    },
    yAxis: {
      type: 'value',
      min: 0,
      axisLabel: {
        formatter: '{value}组'
      }
    },
    dataZoom: [
      {
        type: 'inside',
        start: 0,
        end: 100
      },
      {
        start: 0,
        end: 100
      }
    ],
    series: [{
      data: groupCounts,
      type: 'bar',
      itemStyle: {
        color: new echarts.graphic.LinearGradient(0, 0, 0, 1, [
          { offset: 0, color: '#667eea' },
          { offset: 1, color: '#764ba2' }
        ])
      },
      emphasis: {
        itemStyle: {
          color: new echarts.graphic.LinearGradient(0, 0, 0, 1, [
            { offset: 0, color: '#764ba2' },
            { offset: 1, color: '#667eea' }
          ])
        }
      }
    }]
  };
});

// 方法：加载数据
const loadData = async () => {
  if (!startDate.value || !endDate.value) {
    error.value = '请选择开始和结束日期';
    return;
  }

  if (new Date(startDate.value) > new Date(endDate.value)) {
    error.value = '开始日期不能晚于结束日期';
    return;
  }

  loading.value = true;
  error.value = '';

  try {
    const token = localStorage.getItem('token');
    if (!token) {
      error.value = '请先登录';
      return;
    }

    const response = await axios.get('http://localhost:3001/api/stats/action-groups', {
      params: {
        start_date: startDate.value,
        end_date: endDate.value
      },
      headers: {
        Authorization: `Bearer ${token}`
      }
    });

    actionStats.value = response.data.data;
  } catch (err) {
    error.value = err.response?.data?.message || '加载数据失败';
  } finally {
    loading.value = false;
  }
};

// 方法：排序
const sortBy = (field) => {
  if (sortField.value === field) {
    sortOrder.value = sortOrder.value === 'asc' ? 'desc' : 'asc';
  } else {
    sortField.value = field;
    sortOrder.value = 'desc';
  }
};

// 初始化
onMounted(() => {
  loadData();
});
</script>
```

### 后端API代码

```go
// backend/controllers/stats_controller.go

// GetActionGroupsStats 获取动作组数统计数据
func GetActionGroupsStats(ctx *gin.Context) {
	username, exists := ctx.Get("username")
	if !exists {
		ctx.JSON(http.StatusUnauthorized, gin.H{"error": "User not authenticated"})
		return
	}

	var user models.User
	if err := global.Db.Where("username = ?", username).First(&user).Error; err != nil {
		ctx.JSON(http.StatusInternalServerError, gin.H{"error": err.Error()})
		return
	}

	startDate := ctx.DefaultQuery("start_date", time.Now().AddDate(0, 0, -30).Format("2006-01-02"))
	endDate := ctx.DefaultQuery("end_date", time.Now().Format("2006-01-02"))

	var stats []struct {
		ActionName string `json:"action_name"`
		Groups     int    `json:"groups"`
		TotalReps  int    `json:"total_reps"`
	}

	// 统计每个动作的组数和总次数
	if err := global.Db.Raw(`
		SELECT 
			fa.action_name,
			COUNT(fag.id) as groups,
			SUM(fag.rep_num) as total_reps
		FROM 
			articles a
		JOIN 
			fitness_actions fa ON a.id = fa.article_id
		JOIN 
			fitness_action_groups fag ON fa.id = fag.fitness_action_id
		WHERE 
			a.user_id = ? AND a.created_at BETWEEN ? AND ?
		GROUP BY 
			fa.action_name
		ORDER BY 
			groups DESC
	`, user.ID, startDate, endDate).Scan(&stats).Error; err != nil {
		ctx.JSON(http.StatusInternalServerError, gin.H{"error": err.Error()})
		return
	}

	ctx.JSON(http.StatusOK, gin.H{"data": stats})
}
```

### 路由配置

```go
// backend/router/router.go

func SetupRouter() *gin.Engine {
	r := gin.Default()

	// 认证中间件
	authMiddleware := middlewares.AuthMiddleWare()

	// API路由组
	api := r.Group("/api")
	{
		// 认证相关
		auth := api.Group("/auth")
		{
			auth.POST("/register", controllers.Register)
			auth.POST("/login", controllers.Login)
		}

		// 训练记录相关
		articles := api.Group("/articles", authMiddleware)
		{
			articles.GET("", controllers.GetArticles)
			articles.POST("", controllers.CreateArticle)
			articles.GET("/:id", controllers.GetArticle)
			articles.PUT("/:id", controllers.UpdateArticle)
			articles.DELETE("/:id", controllers.DeleteArticle)
		}

		// 点赞相关
		likes := api.Group("/likes", authMiddleware)
		{
			likes.POST("", controllers.CreateLike)
			likes.DELETE("", controllers.DeleteLike)
		}

		// 评论相关
		api.GET("/articles/:id/comments", controllers.GetCommentsAndReplies)
		api.POST("/articles/:id/comments", authMiddleware, controllers.CreateComment)
		api.POST("/comments/:commentId/replies", authMiddleware, controllers.CreateReply)
		api.DELETE("/comments/:commentId", authMiddleware, controllers.DeleteComment)
		api.DELETE("/replies/:replyId", authMiddleware, controllers.DeleteReply)

		// 统计相关
		stats := api.Group("/stats", authMiddleware)
		{
			stats.GET("/action-groups", controllers.GetActionGroupsStats)
		}
	}

	return r
}
```

## 功能说明

1. **时间范围选择**：用户可以选择开始和结束日期，点击"应用"按钮加载对应时间段的数据。

2. **动作组数统计**：使用柱状图展示每个动作的训练组数，颜色渐变增强视觉效果，支持鼠标悬停查看详细数据。

3. **数据详情表格**：展示每个动作的名称、训练组数和总次数，支持按训练组数或总次数排序。

4. **响应式设计**：页面在不同设备尺寸下都能良好显示，在小屏幕上自动调整布局。

5. **状态管理**：包含加载状态、错误状态和空状态，提升用户体验。

## 技术要点

- **前端**：使用Vue 3组合式API，ECharts图表库，响应式设计。
- **后端**：使用Go语言，GORM ORM库，SQL聚合查询统计数据。
- **API设计**：RESTful风格，支持时间范围参数。
- **用户体验**：流畅的加载动画，清晰的错误提示，直观的数据展示。

通过此页面，用户可以清晰地了解自己在指定时间段内的训练量分布，帮助用户优化训练计划，提高训练效果。
